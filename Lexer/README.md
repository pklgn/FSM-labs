## Другие виды токенов
* Арифметические
    + `+`  сложение
    + `-`  вычитание
    + `*`  умножение
    + `/`  целочисленное деление
    + `**` возведение в степень

* Присваивание
    + `=`

* Сравнение
    + `==` равенства
    + `!=` не равно
    + `<`  меньше
    + `>`  больше
    + `<=` меньше или равно
    + `>=` больше или равно

* Логические операторы
    + `&&` и
    + `||` или

* Побитовые операции
    + `&` и
    + `|` и

* Строка
    + `""` строковый литерал

* Комментарии
    + `//` комментарий до конца строки
